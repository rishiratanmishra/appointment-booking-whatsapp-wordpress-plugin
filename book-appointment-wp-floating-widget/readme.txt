Adds a floating appointment button with a customizable modal form. Sends submissions to WhatsApp, stores in DB, and emails admins.

== Description ==
This plugin adds a floating booking widget to your site. When clicked, it opens a mobile-friendly modal with a customizable form. Submissions are saved to the WordPress database, sent to admin email(s), and open a pre-filled WhatsApp chat to your configured number.

== Features ==
* Floating button left/right, built-in icons, optional animated messages
* Customizable form fields (types: text, textarea, number, email, select, checkbox, date, time, datetime-local, tel)
* Stores submissions in custom table and allows export to CSV
* Emails admin(s) on each submission
* WhatsApp integration via wa.me link

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin
3. Go to Admin → Book Appointments → Settings to configure
